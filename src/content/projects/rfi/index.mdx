Radio frequency interference (RFI) is any unwanted signal within a telescope's operating range.
For McGill's Cosmic Dawn Group, I used a simple machine learning model to create an RFI-identification mechanism,
which serves as a proof of concept for a crucial step in the data processing pipeline.
